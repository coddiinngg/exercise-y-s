# Jawline Challenge UI (Flutter)

6개 참고 화면 스타일을 기준으로 구성한 Flutter UI 프로토타입입니다.

## Run

1. Flutter SDK 설치
2. 프로젝트 루트에서 실행:

```bash
flutter pub get
flutter run
```

## Structure

- `lib/main.dart`: 전체 화면 구현
  - 탭 1: 챌린지 홈(30/60/120일 슬라이더 + Day 카드 + 결제 잠금)
  - 탭 2: Streak + 캘린더
  - 탭 3: Custom 플랜 카드
  - 탭 4: Progress 잠금 프레임
